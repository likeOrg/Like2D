A cozy way to make 2D browser games

Like2D is a lightweight web game framework in the same family as LÖVE or Raylib, but built specifically for the browser. It provides simple bindings to web APIs so you can get things on screen instantly, without fussing with clunky vanilla JavaScript.

Whether you're making a quick prototype or a full-featured 2D game, Like2D gives you the tools you need while staying out of your way. It's a framework, not an engine—giving you the freedom to build how you want.

Why Choose Like2D?

🔥

Fire-and-forget Assets

Graphics and audio that pretend to be synchronous. Load and use assets without callbacks or promises.

🎯

DWIM Graphics

Turns repetitive draw calls into one while removing state bleed for properties like lineCap.

🖼️

Pixel or smooth?

Auto-resize for sharp graphics at any resolution, or pixel-perfect retro style with prescaling.

⭕

Easier Geometry

Vector2 and Rect are just number tuples with a pure-functional library for easy manipulation.

🚲

Consistent APIs

Colors 0-1, not 0-255. Seconds, not milliseconds. Physical buttons, not "A" or "B".

🐦

Light & Elegant

Zero dependencies, less than 5000 lines of code, under 100KiB compressed.

Quick Start

main.ts 
import { createLike } from "like2d";

const like = createLike(document.body);

like.load = () => {
  like.canvas.setMode([800, 600]);
  like.input.setAction("jump", ["Space", "BBottom"]);
};

like.update = (dt) => {
  if (like.input.justPressed("jump")) {
    console.log("Jump!");
  }
};

like.draw = () => {
  like.gfx.clear([0.1, 0.1, 0.1, 1]);
  like.gfx.circle("fill", "dodgerblue", [400, 300], 50);
  like.gfx.print("white", "Hello Like2D!", [20, 20]);
};

await like.start();
