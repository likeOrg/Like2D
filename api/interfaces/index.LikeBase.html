    OptionalhandleEvent

    handleEvent?: TopLevelEventHandler

    LIKE's runtime is built around calling handleEvent.

    This function recieves all events. If set to undefined,

    • Like.callOwnHandlers is the default behavior.

    Otherwise, you can really customize LIKE by setting this to a custom handler.

    For example, the scene architecture is built around setting this function. Setting it to a custom function will disable the scene system.
