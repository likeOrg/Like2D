This is a minimal Javascript starter for [LÏKE.](https://www.npmjs.com/package/@like2d/like)

Open index.html in your browser directly or to enable hot reloading, run
a command like:
```
npx live-server
```

For more serious projects, replace the top import from main
with `@like2d/like` and turn this into an npm package.

Run:
```
npm init
npm i @like2d/like
```

And then change the top line of this js file:
``` typescript
import { createLike, Vec2 } from "@like2d/like"
```
